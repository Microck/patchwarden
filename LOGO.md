# Logo Prompts for Jarspect

**Project Context**: A security scanner for Minecraft mods (.jar files) that flags suspicious behavior before installation. Uses multi-agent analysis (static, behavioral, reputation) to detect malware and protect gamers.

**Suggested Vibe**: Technical, trustworthy, protective, modern security tool

**Color Palette** (from existing UI):
- Primary: `#0a7a6b` (teal/mint)
- Strong: `#045247` (dark teal)
- Danger: `#a53620` (warning red)
- Neutral: `#f5f2eb` (warm cream)

## Usage Notes

1. **Primary color**: Use `#0a7a6b` (teal) for single-color applications
2. **Dark backgrounds**: Use white or `#f5f2eb` for contrast
3. **Danger state**: Logo can appear in `#a53620` for malware warnings
4. **Minimum size**: Ensure legibility at 24px for favicon/app icon
5. **Favicon**: Lettermark (Prompt 4) works best at small sizes
6. **Social/Marketing**: Emblem (Prompt 6) for profile images
